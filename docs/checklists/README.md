# Remote Codex 开发 Checklist 归档

本目录归档 Remote Codex 第一版开发过程中的阶段 checklist。它们保留实现记录、
自动化验证结果与待真实环境执行的验收项，不能替代生产发布或网络验收。

## 阅读顺序

1. [基础协议](01_远程隧道基础协议_checklist.md)：共享协议、配置、流生命周期和身份 capability。
2. [服务端与多用户授权](02_远程隧道服务端与多用户授权_checklist.md)：WSS 中继、共享 agent 授权、配额和审计。
3. [代理端与端到端集成](03_远程隧道代理端与端到端集成_checklist.md)：edge HTTP CONNECT、egress TCP 拨号与联调。
4. [部署与最终验收](04_远程隧道部署与最终验收_checklist.md)：部署资料、真实网络验证、故障演练和发布门禁。

生产操作请使用 [运维文档索引](../operations/README.md)。
